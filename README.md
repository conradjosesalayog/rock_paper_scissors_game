# Rock Paper Scissors (Python)

##  Project Description

This project is a simple **Rock, Paper, Scissors** game developed in **Python**. It allows the user to play against the computer through a graphical user interface (GUI) built with **Tkinter**. The project demonstrates fundamental programming concepts such as user input handling, randomization, conditional statements, and GUI development.

This repository was uploaded as part of our **Object-Oriented Programming (OOP) Final Project**. It is used to analyze and discuss the four pillars of OOP:

* Inheritance
* Polymorphism
* Abstraction
* Encapsulation

The objective of this project is to study an existing Python application and identify how Object-Oriented Programming principles are implemented within its source code. A video presentation accompanies this repository to explain these concepts in detail.
